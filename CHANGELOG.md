# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0]

### Added

- Initial release: `camel_case`, `camel_case_with`, and `camel_case_slice` — a faithful
  port of the `camelcase` npm package, including the `pascalCase`,
  `preserveConsecutiveUppercase`, and `capitalizeAfterNumber` options.
